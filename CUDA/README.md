## To Run:
1. Install python3
2. Install numba
3. set **NUMBA_CUDA_DRIVER** to the location of your libcudart.so, For example ***/share/pkg.7/cuda/10.1/install/lib64/libcudart.so***
(may need to change it in build_graph.py)
4. run `python3 basic_mf.py `
